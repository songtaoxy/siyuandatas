# front-end, back-end, 前端, 后端
{: id="20210410104744-06qapri"}

**前端** （英语：**front-end**）[#terms# ] 和**后端** （英语：**back-end**）[#terms# ]是描述进程开始和结束的通用词汇。前端作用于采集输入信息，后端进行处理。计算机程序的[界面](https://zh.wikipedia.org/wiki/%E7%95%8C%E9%9D%A2 "界面")样式，视觉呈现属于前端。
